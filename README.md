# 🌍 Rihla – Intelligent Tourism Mobile Application

## 📖 Description

**Rihla** is an intelligent mobile tourism application developed as a final-year engineering project. The application helps travelers discover tourist destinations, receive personalized recommendations, explore nearby attractions, and plan their trips using artificial intelligence and geolocation technologies.

The project is built using a modern architecture that combines a mobile application, a RESTful backend, and an AI module to provide a smart and interactive user experience.

---

## ✨ Features

* User authentication and authorization
* Tourist destination discovery
* Smart destination recommendations using AI
* Interactive maps and geolocation
* Route planning
* Search and filtering
* Favorite destinations
* Secure REST API
* Responsive mobile interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/Rihla.git
```

### Backend

```bash
cd backend
```

Configure the database, then run the Spring Boot application.

### Frontend

```bash
cd frontend
npm install
npm start
```

### AI Module

```bash
cd ai
pip install -r requirements.txt
python app.py
```
